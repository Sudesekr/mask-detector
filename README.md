Mask Detector

This project is a computer vision-based application developed to detect whether a person is wearing a mask. It uses image processing and machine learning techniques to analyze faces in real time.

About the Project

Mask Detector is an AI-assisted system designed to identify mask usage on human faces. The application processes camera input and classifies whether a detected face is wearing a mask correctly, not wearing a mask, or wearing it improperly.

Technologies Used
Python
OpenCV
TensorFlow / Keras (or Machine Learning models)
Deep Learning
Computer Vision techniques
Features
Real-time face and mask detection
High accuracy classification
Camera or video stream support
Can be used in public safety scenarios
Lightweight and fast processing
Installation
Download the repository to your computer.
Install the required dependencies (e.g., OpenCV, TensorFlow).
Run the main script using Python.
Allow camera access for real-time detection.
